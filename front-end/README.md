# Fresh Cash Leads Dashboard

Front-end dashboard for browsing Nashville metro area property sales leads. Displays data from 7 Tennessee counties, organized by county tabs with sorting, filtering, search, and CSV export.

Paired with the [Back-End Pipeline](https://github.com/145brice/Property-Managers---Back-End-Recents) which scrapes county assessor portals and generates the CSV data.

## How It Works

The back-end pipeline scrapes public property sales records from county assessor websites, normalizes the data into a standard format, and writes `nashville_cash_leads_clean.csv` directly into this directory. The dashboard reads that CSV on page load.

```
Back-End Pipeline (scrapers)
        |
        v
nashville_cash_leads_clean.csv
        |
        v
index.html (dashboard with county tabs)
```

## Pages

### Dashboard (`index.html`)
- **County Tabs** -- One tab per county (Davidson, Williamson, etc.) plus an "All" tab. Each tab shows only that county's leads with a record count badge.
- **Stats Bar** -- Per-tab summary: total leads, average price, price range, date range
- **Sortable Columns** -- Click Owner Name, Address, Price, or Sold Date to sort
- **Search** -- Real-time text search across all visible fields
- **Price Filter** -- Dropdown with preset ranges ($0-$100K, $100K-$500K, $500K-$1M, $1M+)
- **CSV Export** -- Downloads the current filtered view as a CSV, named by county
- **SOLD Badges** -- Visual indicators on properties with recorded sale dates

### Admin Panel (`admin.html`)
- **County Toggles** -- Enable/disable each of the 7 counties with estimated scrape times
- **Lookback Sliders** -- Configure how many days back each county searches (default: 7 days)
- **Price Range Sliders** -- Set min/max price filter for the pipeline
- **Run Pipeline** -- Starts the back-end scraper with the selected settings
- **Kill Switch** -- Immediately terminates a running pipeline
- **Live Log** -- Real-time pipeline output with status indicator (idle/running/success/error)

### Pricing Page (`pricing.html`)
- Subscription pricing display ($79/month)
- Feature breakdown and FAQ section
- Mock checkout modal (no real payment processing)

### Browse Counties (`counties.html`)
- County selection interface with card layout
- Availability status per county

## Data Format

The CSV has 6 columns:

| Column | Example |
|--------|---------|
| Owner Name | `SMITH JOHN` |
| County | `Davidson` |
| Address | `123 MAIN ST, NASHVILLE, TN` |
| Price | `$350,000` |
| Sold Date | `12/15/2025` |
| Agent Phone | `N/A` |

## Running

### With the back-end server (recommended)
```bash
# From the back-end repo
python server.py
# Dashboard:  http://localhost:8082/
# Admin:      http://localhost:8082/admin.html
```

The server serves this entire front-end directory and provides the pipeline API for the admin panel.

### Standalone (view only, no pipeline controls)
```bash
cd Property-Managers--Front-End
python -m http.server 5020
# Open http://localhost:5020
```

Note: Opening `index.html` directly as a file won't work -- browsers block `fetch()` on `file://` URLs. You need an HTTP server.
